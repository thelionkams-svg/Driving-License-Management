DVLD: Driving & Vehicle License Management System

The Driving & Vehicle License Management System is a professional project designed to manage the full lifecycle of driver licensing and vehicle administration. The project is inspired by a professional reference model and implemented using Three-Tier Architecture and OOP principles, ensuring modularity, maintainability, and clear separation between UI and business logic.

Key Features:

Manage license applications, renewals, replacements, and international permits.

Track applicants and enforce eligibility rules (age, exams, license status).

CRUD operations for people, license classes, and system users with permissions control.

Audit logging to track all user actions and maintain data integrity.
